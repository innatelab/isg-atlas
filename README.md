# The ISG screen analysis
---------
## How to run the jupyter notebook:
1. First you'l need to [install conda](https://conda.io/projects/conda/en/latest/user-guide/install/index.html#regular-installation) depending on your operating system
2. Navigate to the project folder and execute the following command to create an environment and install the necessary packages for this project:

   ```conda env create -f environment.yml```
3. Now you need to activate the evironment using the following command:
   
    ```conda activate isg_analysis```
4. Navigate to the project folder ```cd <path\to\project>```
5. Start jupyter notebook ```jupyter notebook main_analysis.ipynb```
